# ResumeAI – Smart Resume Analyzer

An AI-powered web app that analyzes resumes using the Gemini API.

## Live Demo
https://project-1-13950.web.app

## How to Run
1. Open index.html in your browser
2. Enter your Gemini API key
3. Paste your resume and click Analyze

## Tech Stack
- Gemini 2.5 Flash API
- Firebase Hosting
- HTML, CSS, JavaScript
